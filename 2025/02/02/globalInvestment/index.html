从零开始布局你的海外资产

发表于2025-02-02|更新于2025-02-22|普通人如何赚到一千万
|字数总计:839|阅读量:|评论数:

​ 大家新年好啊，最近小红书上美股投资的话题比较火，如果你也有意向加入海外投资的队伍，就请继续往下读吧。

​ 上图

Wise暂不在本系列教程之内

怎么炒美股

  1. first things first – 开设你自己的香港银行账户（需要办理港澳通行证）

  2. 开通香港/美国券商账户

    鉴于ocbc新加坡开户申请已关闭，目前开通香港本土券商/美国盈透是最优解

    香港常见券商：富途牛牛、长桥Longbridge、盈立香港

  3. 使用香港银行账户入金券商

    通过境内银行电汇，将资金汇到香港银行，详见从内地无损汇款到香港，再将资金通过eDDA等方式汇入券商账户。在内地，这一步叫银证转账

  4. 盈利后直接出金到香港银行账户

怎么炒加密货币

​ 为尽可能规避风险，此方案不包含境内支付方式，即通过微信支付、支付宝、境内银行卡等方式直接通过c2c商家买卖加密货币

  1. first things first – 开设你自己的离岸银行账户

    不限于香港，但香港银行门槛最低，最容易开

  2. 将香港银行卡添加到 applepay/googlepay 等电子钱包

    建议在申请到扣账卡后再进行后续操作

    如果你不知道什么是扣账卡，或者不知道如何申请，请移步零门槛申请的香港银行扣账卡&绑定applepay进行消费

  3. 注册交易所账户（需要身份证）

    比较用的多的：Okex、Binance

    另外注册Bybit及Kraken，后续出金需要用到

    如果你是新人注册，可以参考此篇注册常见交易所账号，内含我的邀请码，新人填写后再入金有一定额度的返现

  4. 通过 applepay/googlepay (相对安全地) 从平台购买usdt（俗称u）

​ USDT（Tether）是一种稳定币，它的价值通常和美元挂钩，也就是说，1 USDT的价值大致等于1美元。它的主要目的是在加密货币市场中提供一个稳定的价值单位，因为比特币和其他加密货币的价格波动较大，而USDT的价值相对稳定。

​ 在加密货币交易所，USDT通常是一个很常见的交易对，可以作为其他加密货币的“中介”。例如，你可以先把法币换成USDT，然后再用USDT购买比特币、以太坊等加密货币。这比直接用法币购买加密货币更加快捷和方便。

​ 我相信大多数人都是通过c2c商家直接微信/支付宝转账买币的，一般c2c卖家的汇率会比较好，但为了规避风险，个人更倾向于从平台买币

  1. 将加密货币兑换成法币

    详见bybit无门槛u卡申请与使用

    及使用kraken及bybit card进行相对安全的出金

    入门的教程就到此了。具体怎样操作，各种交易策略等等，需要各位长期自主学习。

    本系列教程不构成投资建议。由本教程间接构成的任何经济损失本人概不负责。
